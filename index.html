Manga Bot

An amazing tool aims to download manga from the world

This tool aims to download manga from the internet to see them offline later. And certainly, it’s 100% FREE FOREVER ^^. Feel free to feedback to me any errors and new features you want via email dangduy2910@gmail.com or leave comments here. Thanks for using it. Some features:
  • Ability to download a manga from an url
  • Queue to download multiple manga at the same time
  • Automatic create a ZIP or PDF file for every chapter was downloaded complete
  • Turn off computer when queue done
  • Advanced search allow to find manga by its name or url over multiple sites
  • Automatically update manga list
Supported sites:
  • Vietnamese:
    • blogtruyen.com
    • vechai.info
    • manga24h.com
    • truyentranhtuan.com
    • truyentranh8.net
    • izmanga.com
    • otakufc.com
    • truyen.academyvn.com
    • lhmanga.com
  • English:
    • mangafox.me
    • mangapark.me
